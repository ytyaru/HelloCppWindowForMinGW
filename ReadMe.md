# このソフトウェアについて #

HelloC++WindowForMinGWは私個人が学習目的で作成したソフトウェアである。

MinGWのg++でHelloWorldした。（WindowApplication版）

# 開発環境 #

* Windows XP Pro SP3 32bit
* chocolatey
    * g++ (i686-posix-sjlj-rev0, Built by MinGW-W64 project) 5.3.0
